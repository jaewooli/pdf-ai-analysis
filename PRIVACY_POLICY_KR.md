# 개인정보 처리방침 (Privacy Policy)

본 "PDF AI Analyzer Pro" 확장 프로그램(이하 "본 확장 프로그램")은 사용자의 개인정보를 소중히 여기며, 관련 법령을 준수하기 위해 노력하고 있습니다.

## 1. 수집하는 데이터 항목 및 수집 방법

본 확장 프로그램은 다음과 같은 정보를 수집하거나 처리합니다.

- **PDF 문서 내용**: 사용자가 '분석 시작' 버튼을 눌렀을 때, 현재 열려 있는 PDF 파일의 텍스트 또는 파일 자체(Base64 인코딩 형태)를 추출하여 분석을 위해 사용합니다.
- **AI API 키**: 사용자가 설정 화면에서 입력한 OpenAI, Google (Gemini), DeepSeek 등의 API 키를 수집합니다.
- **분석 이력**: 사용자가 수행한 분석의 결과(요약 내용 등)를 저장합니다.

## 2. 데이터의 이용 목적

수집된 데이터는 오직 다음과 같은 목적으로만 사용됩니다.

- **AI 분석 기능 제공**: 추출된 PDF 내용을 사용자가 선택한 AI 서비스(Ollama, OpenAI, Gemini, DeepSeek)로 전송하여 요약 및 분석 결과를 생성합니다.
- **설정 및 기록 유지**: API 키와 과거 분석 이력을 저장하여 사용자가 확장 프로그램을 다시 사용할 때 편리하게 이용할 수 있도록 합니다.

## 3. 데이터의 제3자 제공

본 확장 프로그램은 사용자가 선택한 AI 서비스 제공업체에 PDF 데이터를 전송합니다.

- **제공받는 자**: OpenAI (미국), Google (미국), DeepSeek (중국), 또는 사용자의 로컬 환경(Ollama 등).
- **제공 항목**: PDF 추출 텍스트 또는 PDF 파일 데이터.
- **주의 사항**: 각 AI 서비스 제공업체의 개인정보 처리방침은 해당 업체의 홈페이지에서 확인하시기 바랍니다.

## 4. 데이터의 보관 및 보안

- **로컬 저장**: 본 확장 프로그램은 사용자의 데이터를 별도의 중앙 서버에 저장하지 않습니다. 모든 데이터(API 키, 분석 이력 등)는 사용자의 브라우저 로컬 저장소(`chrome.storage.local`)에만 저장됩니다.
- **보안**: API 키는 외부 서버로 전송되지 않으며, 오직 해당 AI 서비스와의 통신 시에만 사용됩니다.

## 5. 사용자의 권리 및 관리 방법

사용자는 언제든지 자신의 데이터를 관리할 수 있습니다.

- **기록 삭제**: 확장 프로그램의 사이드 패널 내 'Clear History' 버튼을 통해 저장된 모든 분석 이력을 즉시 삭제할 수 있습니다.
- **설정 변경/삭제**: 설정 화면에서 입력된 API 키를 수정하거나 삭제할 수 있습니다.
- **확장 프로그램 삭제**: 브라우저에서 본 확장 프로그램을 삭제하면 로컬에 저장된 모든 데이터도 함께 삭제됩니다.
