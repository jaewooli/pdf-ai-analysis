# Privacy Policy for PDF AI Analyzer Pro

"PDF AI Analyzer Pro" (the "Extension") values your privacy and strictly adheres to the Google Chrome Web Store User Data Policy, including the **Limited Use** requirements.

## 1. Information Collection and Categories
We process the minimum amount of data necessary to provide our services, categorized as follows:
- **User Generated Content**: Text and file data extracted from PDFs for analysis.
- **Authentication Information**: API keys provided by the user (OpenAI, Gemini, DeepSeek).
- **Usage Data**: Locally stored analysis history and results.

## 2. Use of Information & Limited Use Disclosure
The data is processed solely for the primary purpose of **"Providing AI-powered PDF analysis and summaries."**
- We **do not sell** your data to third parties.
- We **do not use** your data for personalized advertising or marketing.
- We **do not use** your data to train AI models without explicit user consent.

## 3. Third-Party Data Sharing (AI Providers)
Depending on your settings, analysis data is sent to:
- **Recipients**: OpenAI, Google, DeepSeek, or your local environment (Ollama).
- **Security**: All data transfers are performed over secure **HTTPS** encrypted connections.

## 4. Data Storage, Security, and Retention
- **Local-Only Storage**: We do not operate any central servers. All sensitive data (API keys, history) is stored exclusively in your browser's local storage (`chrome.storage.local`).
- **Retention**: Data is kept only as long as the Extension is installed. Users can manually delete all data using the "Clear History" button or by uninstalling the Extension.

## 5. User Rights and Controls
You have the full right to access, modify, or delete your API keys and analysis results at any time via the Extension's settings panel.

## 6. Permissions and Their Use
To provide our services, the Extension requires the following permissions:
- **`tabs` & `webNavigation`**: Used to identify PDF files in your browser tabs and provide the analysis interface.
- **`<all_urls>`**: Necessary to detect and access PDF documents across various websites for analysis.
- **`storage`**: Used to securely save your API keys and analysis settings locally on your device.
- **`sidePanel`**: Used to provide a seamless user interface for AI analysis alongside your documents.
