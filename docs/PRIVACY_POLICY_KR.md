# 개인정보 처리방침 (Privacy Policy)

본 "PDF AI Analyzer Pro" 확장 프로그램(이하 "본 확장 프로그램")은 사용자의 개인정보를 소중히 여기며, Google Chrome 웹 스토어의 사용자 데이터 정책을 준수합니다.

## 1. 수집하는 데이터 항목 및 범주
본 확장 프로그램은 서비스 제공을 위해 최소한의 데이터만을 처리하며, 구글의 데이터 범주에 따라 다음과 같이 분류됩니다.
- **사용자 콘텐츠 (User Content)**: 분석 대상이 되는 PDF 문서의 텍스트 및 파일 데이터.
- **인증 정보 (Authentication Information)**: 사용자가 직접 입력한 AI 서비스용 API 키 (OpenAI, Gemini, DeepSeek).
- **사용 기록 (Usage History)**: 로컬에 저장되는 분석 이력 및 결과물.

## 2. 데이터 이용 목적 및 제한적 사용 (Limited Use)
수집된 데이터는 오직 **"PDF 문서의 AI 분석 및 요약 결과 제공"**이라는 단일 목적을 위해서만 사용됩니다.
- 본 확장 프로그램은 수집한 데이터를 제3자에게 판매하지 않습니다.
- 수집된 데이터를 광고 목적으로 사용하지 않습니다.
- 사용자의 동의 없이 모델 학습(Training)용으로 데이터를 활용하지 않습니다.

## 3. 데이터의 제3자 제공 (AI 서비스 제공업체)
사용자가 선택한 AI 엔진에 따라 분석 데이터가 다음 업체로 전송될 수 있습니다.
- **제공처**: OpenAI, Google, DeepSeek, 또는 로컬 LLM (Ollama 등).
- **데이터 보안**: 모든 데이터 전송은 HTTPS 암호화 프로토콜을 통해 안전하게 이루어집니다.

## 4. 데이터 보관 및 보안
- **로컬 저장 방식**: 본 확장 프로그램은 중앙 서버를 운영하지 않습니다. 모든 API 키와 분석 이력은 사용자의 브라우저 로컬 저장소(`chrome.storage.local`)에만 암호화되어 저장됩니다.
- **데이터 보존 기간**: 확장 프로그램을 삭제하거나 사용자가 직접 'Clear History'를 실행할 경우 데이터는 즉시 영구 삭제됩니다.

## 5. 사용자의 권리
사용자는 언제든지 설정 메뉴를 통해 자신의 API 키를 삭제하거나, 모든 분석 기록을 초기화할 권리를 가집니다.

## 6. 권한 사용 및 목적
본 확장 프로그램은 원활한 서비스 제공을 위해 다음 권한을 사용하며, 명시된 목적 외에는 활용하지 않습니다.
- **`tabs` 및 `webNavigation`**: 브라우저 탭 내의 PDF 파일을 식별하고 분석 화면을 제공하기 위해 사용됩니다.
- **`<all_urls>`**: 다양한 웹사이트에 위치한 PDF 문서를 감지하고 분석 데이터를 읽어오기 위해 반드시 필요합니다.
- **`storage`**: 사용자의 API 키 및 개인 설정을 브라우저 로컬 저장소에 안전하게 보관하기 위해 사용됩니다.
- **`sidePanel`**: 문서와 함께 AI 분석 결과를 확인할 수 있는 사용자 인터페이스를 제공하기 위해 사용됩니다.
